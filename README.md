## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Navodya52/nishadi-portfolio.git
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build production version

```bash
npm run build
```
